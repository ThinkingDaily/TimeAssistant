# 周边依赖

Logstash可以单独运行，调试完毕之后，再通过elasticsearch插件和Elasticsearch对接。

# Installation

- 下载tar.gz: https://www.elastic.co/cn/downloads/logstash
- 解压缩tar.gz
- 运行bin/logstash
